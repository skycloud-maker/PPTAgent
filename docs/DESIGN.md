# DESIGN.md

## 목적

이 문서는 PPTAgent가 생성하는 회사 내부 발표자료의 기본 시각 규칙을 정의합니다. 현재 프로젝트는 범용 슬라이드 생성기가 아니라, 내부 보고와 의사결정 문서를 빠르게 만드는 회사 전용 템플릿을 우선 목표로 합니다.

## 기본 방향

- 용도: 회사 내부 보고, 기획안, 상태 공유
- 문체: 간결하고 판단 중심
- 슬라이드 비율: 16:9
- 기본 언어: 한국어
- 기본 폰트: `Malgun Gothic`, `Arial`

## 기본 색상

- Primary: `#A50034`
- Secondary: `#3B455A`
- Accent: `#E9D6DE`
- Text: `#2C2C2C`
- Muted: `#767676`
- Border: `#D9D9D9`
- Surface: `#F8F7F9`

## 템플릿 규칙

### title
- 상단 브랜드 컬러 배너 사용
- 제목은 크게, 부제는 한 줄 보조 정보로 배치
- 발표자 또는 작성자 정보는 우하단 정렬

### section
- 좌측 컬러 패널로 장 구분
- 섹션 제목은 한눈에 보이도록 단문 사용

### bullet
- 제목 영역과 본문 영역을 분리
- 불릿은 최대 5개
- 각 불릿은 1문장 또는 2문장 이내 권장

### two_column
- 비교, 대안 검토, 전후 변화 설명에 사용
- 좌우 제목은 짧고 대비가 분명해야 함

### chart / table / image
- 현재 MVP는 placeholder 중심
- 추후 실제 시각화와 데이터 매핑을 확장 예정

## 브랜딩 자산

- 로고는 외부 URL 다운로드에 의존하지 않음
- `PPTAGENT_LOGO_PATH`로 로컬 이미지를 지정 가능
- 로고가 없으면 회사명 텍스트를 헤더에 표시
- 기밀 문구는 `PPTAGENT_CONFIDENTIAL_LABEL`로 조정 가능
