## 1. 제품 한 줄 정의

> 자연어 요청과 데이터를 입력받아, 회사용 PPT(.pptx)를 자동 생성하는 AI Agent.  
> 비개발자 팀원 누구나 브라우저에서 바로 사용할 수 있다.

---

## 2. 사용자 페르소나

| 항목 | 내용 |
|------|------|
| 주요 사용자 | 비개발자 팀 동료 (기획, 마케팅, 운영 등) |
| 기술 수준 | 비개발자 — 코드 작성 불가, PPT 작성 경험 있음 |
| 사용 환경 | 사내 브라우저 (Streamlit 웹 UI) |
| 핵심 불편함 | 반복적인 보고 장표를 매번 처음부터 만드는 데 시간 소요 |
| 기대 효과 | 장표 초안을 5분 이내에 생성, 이후 직접 수정 |

## 3. 핵심 사용 시나리오 (Use Cases)

### UC-01. 주간 / 월간 업무 보고
- **상황**: 매주/매월 반복되는 팀 업무 보고 장표
- **입력**: 이번 주 한 일, 다음 주 계획, 이슈 사항 (텍스트 입력 또는 메모 붙여넣기)
- **출력**: 표지 + 이번 주 요약 + 주요 업무 상세 + 다음 주 계획 + 이슈/리스크 슬라이드
- **특징**: 템플릿 고정, 매주 내용만 교체하는 반복 사용

### UC-02. 프로젝트 현황 / 개선 결과 보고
- **상황**: 진행 중인 프로젝트 현황 또는 개선 활동 결과를 경영진/유관부서에 보고
- **입력**: 프로젝트명, 기간, 목표, 현재 진행률, 주요 성과, 리스크
- **출력**: 표지 + 프로젝트 개요 + 진행 현황 (타임라인) + 성과 + 리스크/대응 방안
- **특징**: 타임라인, 진행률 시각화 포함

### UC-03. 신규 기획안 제안
- **상황**: 새로운 아이디어나 사업 기획을 내부에 제안하는 장표
- **입력**: 제안 배경, 목적, 핵심 내용, 기대 효과, 필요 자원 (자유 텍스트)
- **출력**: 표지 + 제안 배경/문제 정의 + 솔루션 + 기대 효과 + 실행 계획 + 마무리
- **특징**: AI가 스토리라인 구성을 주도 (범용 생성 모드)

### UC-04. 데이터 분석 결과 / 개선 방향 보고
- **상황**: 데이터 분석 결과를 시각화하고 인사이트와 개선 방향을 함께 전달
- **입력**: CSV / JSON / Parquet 데이터 파일 + 분석 맥락 설명
- **출력**: 표지 + 분석 배경 + 주요 지표 요약 + 차트 슬라이드 (복수) + 인사이트 + 개선 방향
- **특징**: 데이터 파일 업로드 → 차트 자동 생성, VoC 대시보드 연동 확장 가능

## 4. 기능 목록 (Feature List)

### 4-1. 핵심 기능 (MVP)

| ID | 기능 | 설명 |
|----|------|------|
| F-01 | 자연어 입력 | 사용자가 텍스트로 장표 내용을 자유롭게 입력 |
| F-02 | 템플릿 선택 | UC-01~04에 맞는 사전 정의 템플릿 선택 |
| F-03 | AI 슬라이드 기획 | 입력을 분석해 슬라이드 구조(Slide Schema) 자동 생성 |
| F-04 | 슬라이드 미리보기 | 생성 전 슬라이드 구조 확인 (제목 목록 수준) |
| F-05 | .pptx 생성 및 다운로드 | 최종 파일 생성 후 즉시 다운로드 |
| F-06 | 데이터 파일 업로드 | CSV / JSON 업로드 → 차트 슬라이드 자동 생성 |

### 4-2. 2차 기능 (Post-MVP)

| ID | 기능 | 설명 |
|----|------|------|
| F-07 | 슬라이드 순서/내용 편집 | 생성된 Schema를 UI에서 수정 후 재생성 |
| F-08 | 템플릿 커스터마이징 | 회사 로고, 색상, 폰트 등 브랜드 설정 |
| F-09 | 이전 생성 이력 조회 | 최근 생성한 장표 목록 및 재생성 |
| F-10 | PDF 내보내기 | .pptx 외 PDF 포맷 추가 지원 |
| F-11 | VoC 대시보드 연동 | Parquet 데이터 직접 주입 → 분석 리포트 자동화 |

## 5. 기능 우선순위 (Prioritization)

```
MVP (1차 목표)
├── F-01 자연어 입력
├── F-02 템플릿 선택
├── F-03 AI 슬라이드 기획
├── F-04 슬라이드 미리보기
├── F-05 .pptx 생성 및 다운로드
└── F-06 데이터 파일 업로드

Post-MVP (2차)
├── F-07 슬라이드 편집
├── F-08 템플릿 커스터마이징
└── F-09 이전 생성 이력

장기 (3차)
├── F-10 PDF 내보내기
└── F-11 VoC 대시보드 연동
```

## 6. 비기능 요구사항 (Non-functional Requirements)

| 항목 | 요구사항 |
|------|----------|
| 생성 속도 | 슬라이드 10장 기준 60초 이내 생성 |
| 보안 | 생성 파일 서버 미저장, 로그에 내용 미기록 |
| 접근성 | 브라우저만 있으면 사용 가능 (설치 불필요) |
| 언어 | 한국어 입력/출력 기본 지원 |
| 파일 호환성 | 생성된 .pptx가 MS PowerPoint / Google Slides에서 정상 열림 |
