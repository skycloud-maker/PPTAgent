## 1. 기술 스택

| 항목 | 선택 | 이유 |
|------|------|------|
| 프레임워크 | Streamlit | 기존 VoC 프로젝트와 동일 스택, 빠른 구현 |
| 배포 | Streamlit Cloud | 별도 인프라 불필요 |
| 언어 | Python | 백엔드와 동일 언어로 통합 관리 |

## 2. 전체 화면 흐름 (User Flow)

단계별 위자드(Step-by-step) 방식. 총 4단계.

```
[Step 1] 템플릿 선택
       ↓
[Step 2] 내용 입력
       ↓
[Step 3] 미리보기 확인
       ↓
[Step 4] 생성 & 다운로드
```

- 각 단계는 상단 진행 바(Progress bar)로 현재 위치 표시
- 이전 단계로 돌아가기 항상 가능
- 중간에 이탈해도 입력 내용 세션 내 유지

## 3. 화면별 상세 설계

---

### Step 1 — 템플릿 선택

**목적**: 어떤 종류의 장표를 만들지 선택

**UI 구성**:
```
┌─────────────────────────────────────────────┐
│  어떤 장표를 만들까요?                         │
│                                             │
│  ┌─────────┐ ┌─────────┐ ┌─────────┐       │
│  │ 📋      │ │ 📊      │ │ 💡      │       │
│  │주간/월간 │ │프로젝트  │ │신규     │       │
│  │업무 보고 │ │현황 보고 │ │기획안   │       │
│  └─────────┘ └─────────┘ └─────────┘       │
│                                             │
│  ┌─────────┐                                │
│  │ 📈      │                                │
│  │데이터   │                                │
│  │분석 리포트│                               │
│  └─────────┘                                │
│                                             │
│                      [다음 →]               │
└─────────────────────────────────────────────┘
```

**동작 규칙**:
- 카드 클릭 시 선택 상태(하이라이트) 표시
- 선택 없이 다음 버튼 클릭 시 경고 메시지
- 선택한 템플릿에 따라 Step 2 입력 폼이 다르게 표시됨

---

### Step 2 — 내용 입력

**목적**: 장표에 들어갈 내용을 입력

**UI 구성 (템플릿별로 다름)**:

#### UC-01 주간/월간 업무 보고
```
┌─────────────────────────────────────────────┐
│  내용을 입력해주세요                           │
│                                             │
│  보고 기간 *                                 │
│  [2024년 10월 4주차              ]           │
│                                             │
│  이번 주 주요 업무 *                          │
│  [                               ]          │
│  [  자유롭게 입력하세요.           ]          │
│  [  예) - VoC 대시보드 데이터 업데이트 ]      │
│                                             │
│  다음 주 계획 *                              │
│  [                               ]          │
│                                             │
│  이슈 / 특이사항 (선택)                       │
│  [                               ]          │
│                                             │
│  [← 이전]                      [다음 →]     │
└─────────────────────────────────────────────┘
```

#### UC-04 데이터 분석 리포트 (데이터 업로드 포함)
```
┌─────────────────────────────────────────────┐
│  내용을 입력해주세요                           │
│                                             │
│  분석 제목 *                                 │
│  [2024 Q3 고객 분석 결과         ]           │
│                                             │
│  분석 배경 및 목적 *                          │
│  [                               ]          │
│                                             │
│  데이터 파일 업로드 (CSV / JSON)              │
│  [  파일을 여기에 끌어다 놓으세요  ]          │
│                                             │
│  개선 방향 (선택)                            │
│  [                               ]          │
│                                             │
│  [← 이전]                      [다음 →]     │
└─────────────────────────────────────────────┘
```

**동작 규칙**:
- `*` 표시 필드는 필수 입력
- 필수 필드 미입력 시 다음 단계 진행 불가 + 인라인 오류 표시
- 데이터 파일 업로드 시 컬럼 미리보기 표시 (상위 3행)
- 텍스트 영역은 자동 높이 조절

---

### Step 3 — 미리보기 확인

**목적**: AI가 기획한 슬라이드 구조를 확인하고 승인

**UI 구성**:
```
┌─────────────────────────────────────────────┐
│  슬라이드 구조를 확인해주세요                  │
│                                             │
│  총 7장의 슬라이드가 생성됩니다               │
│                                             │
│  1  표지          제목 / 부제 / 날짜         │
│  2  목차          전체 구성 안내             │
│  3  이번 주 요약   핵심 3가지 불릿            │
│  4  주요 업무 1    상세 내용                 │
│  5  주요 업무 2    상세 내용                 │
│  6  다음 주 계획   일정 및 목표              │
│  7  이슈 / 리스크  현황 및 대응 방안         │
│                                             │
│  ✏️ 구조가 마음에 들지 않으면                 │
│     [← 내용 수정하기]                       │
│                                             │
│  [← 이전]              [장표 생성하기 →]    │
└─────────────────────────────────────────────┘
```

**동작 규칙**:
- 슬라이드 목록은 번호 + 제목 + 한 줄 설명으로 표시 (썸네일 없음, MVP)
- "내용 수정하기" 클릭 시 Step 2로 돌아감 (입력 내용 유지)
- "장표 생성하기" 클릭 시 Step 4로 이동 + 생성 시작

---

### Step 4 — 생성 & 다운로드

**목적**: .pptx 파일 생성 후 다운로드

**UI 구성**:

#### 생성 중 상태
```
┌─────────────────────────────────────────────┐
│                                             │
│           ⏳ 장표를 만들고 있어요...           │
│                                             │
│   [=================·······]  68%           │
│                                             │
│   슬라이드 5/7 생성 중                       │
│                                             │
└─────────────────────────────────────────────┘
```

#### 생성 완료 상태
```
┌─────────────────────────────────────────────┐
│                                             │
│           ✅ 장표가 완성됐어요!               │
│                                             │
│   📄 2024_10월4주차_업무보고.pptx            │
│      슬라이드 7장                            │
│                                             │
│   [⬇️ 다운로드]                             │
│                                             │
│   ────────────────────────────             │
│   [+ 새 장표 만들기]                        │
│                                             │
└─────────────────────────────────────────────┘
```

**동작 규칙**:
- 생성 중 진행률 표시 (슬라이드 단위)
- 생성 완료 후 자동 다운로드 트리거
- 파일명 자동 생성: `{날짜}_{템플릿명}.pptx`
- 다운로드 완료 후 서버 임시 파일 즉시 삭제
- "새 장표 만들기" 클릭 시 Step 1으로 초기화

## 4. 공통 UI 요소

### 진행 바 (Progress Bar)
```
● ─────── ○ ─────── ○ ─────── ○
1.템플릿   2.내용    3.확인    4.완료
```
- 완료된 단계: 채워진 원 (●)
- 현재 단계: 강조 표시
- 미완료 단계: 빈 원 (○)

### 오류 메시지
- 필수 입력 누락: 해당 필드 아래 빨간 텍스트
- API 오류: 페이지 상단 오류 배너 + 재시도 버튼
- 파일 업로드 실패: 인라인 오류 메시지

### 로딩 상태
- AI 기획 중 (Step 2 → 3): "슬라이드 구조를 기획하는 중이에요..." 스피너
- 파일 생성 중 (Step 3 → 4): 진행률 바
